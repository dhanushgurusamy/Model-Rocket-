# Model-Rocket-
Design, analysis, build, testing and launch preparation of a model rocket for a 1 km mission, including motor design, airframe design, avionics, cameras, recovery, and ground station work.

## Overview
This project covers the design, development, testing, and launch preparation of a student model rocket.  
The work includes motor design, nozzle design, casing and end cap design, airframe design, avionics, telemetry, recovery and ground station development.

The project was developed at SPARC AEROTECH PVT LTD with a focus on safe flight, stable performance, and reliable data collection.  
It was built using a mix of design, simulation, fabrication, and testing work.

## Project Goals
- Design a stable and reliable model rocket system
- Develop the motor and major motor components
- Prepare the airframe and structural parts
- Add avionics and telemetry support
- Ensure safe recovery after flight
- Validate the design through testing and simulation

## Key Work Done
- Selected the propellant and grain structure for the motor
- Designed the nozzle using simulation and CAD
- Designed the end cap and casing for the motor
- Worked on rocket body design and system integration
- Planned telemetry and ground station communication
- Designed the recovery system for safe descent
- Prepared test plans and launch support documents

## Tools Used
